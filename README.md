## Tonatiuh2CSV 

This program converts the binary output file of [Tonatiuh](https://github.com/iat-cener/tonatiuh) to a Comma-separated values (.csv) file

Usage: Tonatiuh2CSV inputFile 

Generates a file called inputFile.csv
